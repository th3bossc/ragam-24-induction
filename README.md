## Installation 

1. Clone the repo
  ```bash
    git clone https://github.com/th3bossc/ragam-24-induction.git
  ```
 
2. Change directory and install dependencies
  ```bash
    cd ragam-24-induction
    npm install
  ```

3. Run the server

  ```bash
    npm run dev
  ```

4. The server should be running on http://localhost:3000
